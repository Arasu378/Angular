# Required Software
- Code Editors:
    - **VS Code**
        - Light and fast, with many plugins
    - **Visual Studio**
        - Not so light and fast, still okay
    - Other editors are okay as well
        - They `lack some TS features`, like debugging
- Code configurations will be shown in **VS Code**
